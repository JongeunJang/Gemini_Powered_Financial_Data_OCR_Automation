작업 프로세스 요약 (Master Workflow)

[Step 1] Scouting (선별): Step 1 Prompt + Screenshot of PDFs to Gemini
- 전체 PDF를 훑으며 연도 범위와 스캔 퀄리티를 확인합니다.
입력: 전체 PDF 리스트
Action: 각 PDF의 첫 페이지만 빠르게 읽어서 다음 두 가지를 기록합니다.
커버리지: (예: File_A.pdf -> 1951~1947)
화질 점수: (예: File_A.pdf -> 9점, File_B.pdf -> 4점)
출력 결과: [연도 범위, 화질 점수]가 태깅된 파일 리스트

- 가장 선명한 연도 컬럼을 가진 파일들을 **'마스터 PDF'**로 최종 선정합니다. (중복 연도는 화질이 더 좋은 쪽 하나만 선택)
목표: 1951년 데이터가 필요함.
후보군: 1951년을 포함하는 모든 PDF 검색.
PDF_A (1951~1947) : 화질 9점
PDF_B (1952~1945) : 화질 5점
선택: 화질이 압도적으로 좋은 PDF_A 선택.
확정: PDF_A가 가진 1951, 1950, 1949, 1948, 1947 데이터는 이 파일에서 가져오기로 확정.
다음 목표: 1947년까지 확보했으므로, 1946년을 포함하는 PDF를 찾으러 감.
반복: 1946년을 포함하는 파일 중 최고 화질 파일 선택... (이 과정을 1928년까지 반복)

- 만약 10개 이상이면 계속 업로드하고

-마지막에 Done 치긴	

[Step 2] Extraction (추출): Step 2 Prompt + Prompt V19 to Gemini
- 선정된 마스터 PDF들을 Gemini에게 주고, 아래 제공할 **[전용 프롬프트]**를 입력합니다.
- Gemini가 내뱉는 **JSON 데이터 덩어리(Text)**를 복사합니다. 그리고 별도의 txt파일에 붙여넣고 저장합니다.

[Step 3] Python Execution (생성): JSON to Python
- 실행합니다.
- 각 PDF 이름으로 된 엑셀 파일들이 자동으로 생성됩니다.
